# Fileshipper

Client for Filestack SDK (https://github.com/filestack/filestack-js)

## Setup
* Node should be installed
* Add your Filestack API KEY in config/config.js
```
  $ npm install
```

## Run
```
  $ npm install
  $ npm run serve
```

## Run tests
```
  $ npm run test:unit
```
* The app will start on port 8080.
